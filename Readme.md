### 这个是基于go语言实现的树莓派摄像头的功能
#### 简介
* 使用树莓派的摄像头并一直录制视频

### 前提条件
* 准备一个树莓派(3b及以上)
* 给树莓派安装好一个摄像头
* 运行以下脚本命令:
```shell
curl -o load.sh https://raw.githubusercontent.com/Ledgerbiggg/go-raspberry-camera/main/load.sh
chmod +x load.sh
./load.sh
```
